前言

在现在这个前端开发的时代，sass、less这样的css编译工具以及必不可少了，能够大大提升我们的开发效率，在vue框架流行之时，vue-cli这个项目快速构建工具相信是不少人的选择，vue-loader本身已经支持第三方编译工具，我们只需要稍微配置一下，就能够在vue里面欢快的使用sass了。

验证sass是否生效

这个时候你可以进入项目目录, 在assets下随便创建一个.sass/.scss的文件，然后在main.js进行引用，再给App.vue的 style 标签加上 lang=”scss”。

运行看一看有没有报错，因为最新的vue-cli内部已经支持了scss，如果没有报错你就不需要往下看了，可以开始你的编程了。（之前自己就是升级了vue-cli然后又配置了loader，import一个scss文件一直报错，找了很久才找到是这个问题）

node-sass报错vendor的解决方法

发表于 2017-06-04

解决window下使用npm运行项目时报莫\node_modules\.4.5.3@node-sass\vendor不能找到的错误。

首先是在node_modules这个目录下找到*@node-sass，然后在这个文件夹中添加vendor文件夹，执行npm重装命令。

如果安装失败，会生成一个目录名为类似这样win32-x64-48的空目录，

然后去 https://github.com/sass/node-sass/releases 这个网站下载对应这个目录名的.node文件放到这个目录下即可。

如果报找不到c:/Python.exe的错误，则需要安装python环境，并且在系统环境变量中加上python的安装路径，然后把node_modules目录下的node-sass目录删掉，重装node-sass

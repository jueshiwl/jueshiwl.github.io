git常用方法记录

发表于 2017-05-10

安装配置

安装 git 就不用说了，百度 git 自己安装一下就行了。配置要在命令行输入

git基本操作

1.添加修改文件至暂存区

2.将暂存区文件提交至本地git仓库

3.获取当前仓库的修改状态（例：有readme修改了还没有add之类的状态**

4.查看修改文件的前后差异

5.将本地git仓库中的数据推送至远程

6.拉取远程代码与本地代码进行合并（如有冲突需手动解决

git分支操作

1.创建分支（给当前分支再创建一个分支）

2.切换分支（需确认本地有代码是否还有修改没有提交）

3.合并分支（将选定的分支合并到当前分支上）

4.删除分支（删除选定的分支，不可删除当前所在的分支）

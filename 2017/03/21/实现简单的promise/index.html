实现简单的promise

发表于 2017-03-21

在项目的开发过程中，ES6的使用越来越频繁，而其中的重点promise更是基本占据了我们整个项目，所以就想要对它进行更深入的了解。

让我们先打印一下Promise，正如大家所看到的Promise实际上就是一个构造函数。

我们都知道promise有三种状态（pending,fulfilled,rejected,当resolve执行的时候实际上是把状态值pending变成了 fulfilled，我现在想要来写一个简单的Promise

```
function Promise(fun) {
this.stauts = ‘pending’

fun(_resolve, _reject)

this.prototype.then = function (resolve, reject) {
    this.resolve = resolve
    this.reject = reject
}

function _resolve(data) {
    this.stauts = "success"
    this.resolve(data || '')
}
function _reject(err) {
    this.stauts = "success"
    this.err = err
}

}

new Promise(function(resolve, reject) {
setTimeout(function(){
resolve()
}, 1000)
}).then(function(){
console.log(‘1秒’)
})

一运行就爆了个错误
看了一下应该是自己原型的写法有问题，换了一种 写法

function Promise(fun) {
var self = this

this.stauts = 'pending'

fun(_resolve, _reject)


function _resolve(data) {
    self.stauts = "success"
    self.resolve(data || '')
}
function _reject(err) {
    self.stauts = "success"
    self.resolve(err || '')
}

}

Promise.prototype = {
then: function(resolve, reject) {
this.resolve = resolve || function(){}
this.reject = reject || function(){}
}
}

new Promise(function(resolve, reject) {
setTimeout(function(){
resolve()
}, 1000)
}).then(function(){
console.log(‘1秒’)
})

运行一下发现没问题，一秒后控制台输出了‘1秒’，但是这样也只是实现了简单的resolve，还有就是stauts这个状态根本就没有使用到

首先它是一个构造函数，接受一个函数作为参数，这两个参数也是函数，
